## Features

**Hybrid AI Routing**
- **LOCAL** — Run queries through Ollama models on your machine. Your data never leaves your computer.
- **CLOUD** — Access OpenAI, Anthropic Claude, Google Gemini, and Azure OpenAI for maximum capability.
- **AUTO** — Intelligent routing based on content sensitivity and task complexity.

**Profiles & Knowledge Bases**
- Custom system prompts that shape AI behavior for different contexts
- Attach documents (Markdown, text) as reference knowledge
- Persistent memory across conversations
- Per-profile scratchpads for working notes

**Project Management**
- Organize work around specific goals with persistent chat history
- Roadmaps with goals, phases, and tasks
- AI-powered Roadmap Builder for automatic project planning

**Analytics & Cost Tracking**
- Real-time token usage and cost estimates
- Detailed reports: Token Usage, Usage Over Time, Chat Analytics
- Export to CSV

**Workspace Flexibility**
- Multi-tab interface with detachable windows
- 20+ color themes (dark and light variants)
- Per-tab color customization
- Dockable panels: Local Models, Cloud Models, Analyze Explorer, Execution History, and more

**Security**
- Optional authentication for API key protection
- Supports Windows Hello, YubiKey, and TOTP (Authenticator apps)

**Built-in Tools**
- Workshop for diff/merge operations
- Compare mode for quick text comparison
- Analyze Explorer for file analysis workflows
- Context Inspector for debugging prompts

## System Requirements

- Windows 10 or Windows 11 (64-bit)
- Internet connection (for cloud providers)
- For local models: [Ollama](https://ollama.ai) installed and running

---

## Quick Start

1. **Download** the latest release ZIP and extract to any folder
2. **Run** `HuskyOps.exe`
3. **Complete** the Setup Wizard (configure providers and API keys)
4. **Select** a routing mode (LOCAL, CLOUD, or AUTO) and start chatting

## Supported Providers

| Provider | Models |
|----------|--------|
| **Ollama** | Llama, Mistral, CodeLlama, and any local models |

| **Anthropic** | Claude 3.5 Sonnet, Claude 3 Opus, Claude 3 Haiku |
| **Google** | Gemini Pro, Gemini Pro Vision |
| **Azure OpenAI** | Your deployed Azure OpenAI models |
